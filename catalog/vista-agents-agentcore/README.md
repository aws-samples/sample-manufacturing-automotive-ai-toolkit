# In Vehicle Agentic AI Agents -  AgentCore

A multi agent colloboration framework to improve the in-vehicle experience especially focused on diagnostic trouble codes and service center experience.

## 🚨 Important: Agent Configuration Options

> MA3T contains **two different agent implementations** of In Vehicle Agentic AI Agents:
> 
> - **Current folder**: Agents configured for **Bedrock Agent Core**
> - [**vista-agents**](../vista-agents/README.md): Agents configured for **native Amazon Bedrock**
> 
> Choose the implementation that matches your requirements. Both provide the same functionality but use different underlying architectures.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
