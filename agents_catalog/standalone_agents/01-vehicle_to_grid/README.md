# vehicle_to_grid

The Vehicle-to-Grid (V2G) MCP Server enables AI assistants to access real-time electricity market data and utility rates to provide personalized charging recommendations for electric vehicle owners. By analyzing grid demand, electricity prices, and utility rates, the server can suggest optimal times for charging and discharging to minimize costs and maximize revenue from selling energy back to the grid.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
